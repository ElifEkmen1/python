# 🧮 Python Tkinter Calculator

This is a graphical calculator application developed using Python's **Tkinter** library. The project was created to practice fundamental algorithmic logic and GUI (Graphical User Interface) design.

## ✨ Features

- **Basic Operations:** Addition, subtraction, multiplication, and division.
- **Advanced Functions:** Percentage calculation, toggle positive/negative (+/-).
- **Visual Design:** User-friendly color palette and auto-centered window placement.
- **Logic Handling:** Prevents multiple decimal points and includes a function to clean unnecessary trailing zeros.

## 🚀 Getting Started

To run the project on your local machine:

1. Clone this repository.
2. Ensure you have Python installed on your system.
3. Run the file via terminal or command prompt:
   ```bash
   python python_project_calculator.py
